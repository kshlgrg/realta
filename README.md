# RealTa - Elegant Responsive Real Estate Website

## Project Description

RealTa is a professional, modern, and elegant multi-page real estate listing website designed for showcasing properties, agents, services, testimonials, and company information. Developed as a college capstone project, it aims to deliver a responsive user experience that adapts seamlessly across desktop, tablet, and mobile devices.

Built with clean, semantic HTML5, CSS3, and minimal vanilla JavaScript, RealTa features a trusted Indian real estate brand identity, carefully crafted color palette, and Indian urban user-focused content.

***

## Features

- Responsive layout with flexible grids and media queries for smooth viewing on all devices  
- Deep Navy (#182848) and Goldenrod (#F2C14E) color scheme projecting professionalism and elegance  
- User-friendly navigation with a sticky header and collapsible hamburger menu on mobile  
- 10 fully designed pages: Home, About, Properties, Agents, Gallery, Services, Testimonials, Contact, Blog, Careers  
- Property listings with images, pricing in INR, and location  
- Agent profiles with professional photographs and specialties  
- Interactive and validated contact form  
- Client testimonials with ratings and images  
- Blog section for real estate news and tips  
- Careers page highlighting job openings and company culture  
- All images embedded via high-resolution Unsplash URLs (no downloads required)  
- Clean, comment-free code for easy understanding and explanation in viva  

***

## Technology Stack

- HTML5 (semantic tags)  
- CSS3 (external stylesheet, media queries, Flexbox/Grid)  
- Vanilla JavaScript for menu toggle and form validation  

***

## How to Use

1. Download or clone the project files.  
2. Open `index.html` in any modern web browser.  
3. Navigate through pages using the header menu or internal links.  
4. Test the responsive design by resizing the browser or using developer tools.  
5. Use the contact form on the Contact page to test validation (alerts upon submission).
